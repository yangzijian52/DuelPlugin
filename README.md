               必要条件：利用EssentialsX插件的/setwarp功能创建出生点a，b（必须叫这个名字，吗，否则不好使）
                        创建地标指令为
                        /setwarp a 
                        /setwarp b


               使用方法：放入插件plugins/文件夹内

               指令：/dt     加入匹配队列
                    /dt q  取消匹配


               其他：10分钟自动平局
                      5分钟匹配超时
